# CSV парсер

Простой парсер CSV-файлов на языке Go, написанный в рамках изучения основ языка. Программа считывает данные о сотрудниках, конвертирует их и рассчитывает общую статистику по зарплатам и возрасту.
Использование тем: циклы(for+range), массивы, slices, make/cap, append, strings/runes

## 🚀 Возможности
* Чтение данных из внешнего `.csv` файла.
* Обработка строк с использованием пакета `strings`.
* Конвертация текстовых данных в числовые типы (`strconv`).
* Расчет общих показателей:
    * Общее количество сотрудников.
    * Средний и суммарный возраст.
    * Средняя и суммарная зарплата.

## 🛠 Технологии
* **Go 1.20+**
* Стандартная библиотека: `fmt`, `os`, `strings`, `strconv`, `log`.

## 📋 Формат входных данных
Программа ожидает файл `salary.csv` в корневой папке проекта. Формат файла должен быть следующим (обязательно наличие заголовка):

```csv
name,age,salary
Leo,25,1000
Anna,30,2000
Bob,44,4760
